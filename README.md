# TierRanking

실시간으로 여러 사용자가 공동으로 티어 랭킹을 매길 수 있는 웹 사이트입니다.

## 주요 기능

- 🎯 **실시간 동기화**: Socket.IO를 사용하여 모든 사용자의 변경사항이 실시간으로 동기화됩니다
- 👥 **다중 사용자 지원**: 여러 명이 동시에 접속하여 협업할 수 있습니다
- 🎨 **드래그 앤 드롭**: 직관적인 드래그 앤 드롭 인터페이스로 아이템을 티어에 배치할 수 있습니다
- ➕ **아이템 추가**: 실시간으로 새로운 아이템을 추가할 수 있습니다
- 📊 **5단계 티어 시스템**: S, A, B, C, D 티어로 아이템을 분류할 수 있습니다

## 기술 스택

- **백엔드**: Node.js + Express
- **실시간 통신**: Socket.IO
- **프론트엔드**: HTML5, CSS3, JavaScript

## 설치 및 실행

### 사전 요구사항

- Node.js (v14 이상)
- npm

### 설치

```bash
npm install
```

### 실행

```bash
npm start
```

서버가 시작되면 브라우저에서 `http://localhost:3000`에 접속하세요.

## 사용 방법

1. 여러 브라우저 창 또는 여러 기기에서 사이트에 접속합니다
2. "새 아이템 이름을 입력하세요" 입력 필드에 아이템 이름을 입력하고 "아이템 추가" 버튼을 클릭합니다
3. 미배치 영역에 추가된 아이템을 드래그하여 원하는 티어(S, A, B, C, D)에 드롭합니다
4. 모든 변경사항이 실시간으로 모든 접속자에게 공유됩니다
5. 우측 상단에서 현재 접속자 수를 확인할 수 있습니다

## 티어 설명

- **S 티어**: 최상위 등급 (빨간색)
- **A 티어**: 상위 등급 (노란색)
- **B 티어**: 중상위 등급 (파란색)
- **C 티어**: 중하위 등급 (녹색)
- **D 티어**: 하위 등급 (보라색)
- **미배치**: 아직 티어가 배정되지 않은 아이템 (회색)
